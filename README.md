Chatbot UI

Interactive Multilingual Chatbot UI with Hindi, English and Marathi support, dynamic FAQ questions, automated responses and modern chat interface.

Features
Multilingual support (Hindi, English, Marathi)
Dynamic question buttons
Automated FAQ responses
Interactive chat interface
Clean and modern UI design
Scrollable chat window
Language selection options
Technologies Used
HTML
CSS
JavaScript
Functionality
User selects preferred language
Bot displays predefined questions
User clicks question buttons
Bot returns automated answers
After completion, help options appear
Use Case

This chatbot UI can be used for:

OTT platform support chatbot
Website FAQ chatbot
Customer support UI
TV app assistant UI
